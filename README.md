
# 🐍 Python POO Exercises — Do Básico ao Avançado

Este repositório reúne exercícios práticos de **Programação Orientada a Objetos (POO) em Python**, organizados de forma progressiva — do básico ao avançado.

O objetivo é consolidar fundamentos essenciais para aplicações reais, com foco em **lógica, organização de código e boas práticas**.

---

## 🚀 Objetivo

Desenvolver domínio em:

- Criação de classes e objetos
- Métodos e atributos
- Encapsulamento
- Regras de negócio
- Herança e reutilização de código
- Estruturação de sistemas simples

---

## 📚 Estrutura do Projeto

├── nivel_1/ # Fundamentos de classes
├── nivel_2/ # Métodos e regras
├── nivel_3/ # Encapsulamento
├── nivel_4/ # Herança e abstração
└── LeetCode - Final Level/ # Problemas mais completos


---

## 🧩 Níveis de Exercícios

### 🟢 Nível 1 — Básico
Introdução à criação de classes, atributos e métodos simples.

Exemplos:
- Produto com cálculo de desconto
- Pessoa com verificação de idade
- Contador simples

---

### 🟡 Nível 2 — Intermediário
Aplicação de regras de negócio e manipulação de dados.

Exemplos:
- Conta bancária (depósito e saque)
- Sistema de notas de aluno
- Carrinho de compras

---

### 🟠 Nível 3 — Encapsulamento
Controle de acesso e proteção de dados.

Exemplos:
- Conta bancária com saldo protegido
- Controle de estoque com validações

---

### 🔴 Nível 4 — Avançado
Uso de herança e modelagem de sistemas.

Exemplos:
- Sistema de funcionários (com bônus)
- Sistema de veículos
- Estrutura de pedidos

---

### 🧠 Desafios
Problemas mais próximos do mundo real.

Exemplo:
- Sistema de biblioteca (empréstimo de livros)

---
